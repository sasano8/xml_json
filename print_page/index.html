    example

    These codes are roughly equivalent. Not fully compatible, but can convert structures to each other.

    @values(
    "text1"
    @values("text2")
    "text3"
)

    <values>
    text1
    <values>text2</values>
    text3
</values>

    (values
    (value "text1")
    (values (value "text2"))
    (value "text3")
)

    {"values": [
    "text1,
    {"values": "text2"},
    "text3"
]}

    ["values",
    ["value", "text1"],
    ["values", ["value", "text2"]],
    ["value", "text3"]
]

    node chain

    supports chaining of nodes, Supports node chaining and achieves beautiful node representation. Binary tree representation will be cleaner.

        1
    :@add(2)
    :@add(3)

        @add(
        @add(
            1
            2
        )
        3
    )

    # 型にするか/キーバリューにするか
@node(
    name: aaa

)

# 構文を分ける
@node(
    elem_type = 1
    @shape(
        @dim(
            dim_value = 1
        )
    )
